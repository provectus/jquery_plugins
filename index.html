Selects

Description:
Plugin replaces default selects with more beautiful analogues. It monitors changes in original element and applies them on itself.
Usage:
<select class="s-select class-to-find" placeholder="placeholder">
  <optgroup label="Number Values">
   <option>Value 1</option>
   <option>Value 2</option>
  </optgroup>
  <option>Value A</option>
  <option>Value B</option>
</select>
$(function () {
  //initialize
  $('select.class-to-find').selects(options);
  //append item
  $('select.class-to-find').append('<option>Value</option>');
  //remove element
  $('select.class-to-find').find('<selector-of-option>').remove()
  //getting value
  $('select.class-to-find').val();
  //setting value
  $('select.class-to-find').val('<value of selecting option>');
  //disable
  $('select.class-to-find').prop('disabled', true);
  //enable
  $('select.class-to-find').prop('disabled', false);
});
Options:
{
  input: true/false // text field in select, default: false
}

Datepicker

Description:
Plugin draws pretty calendar for choosing date.
Usage:
<input type="text" class="s-calendar class-to-find" />
$(function () {
  //initialize
  $('input.class-to-find').datepicker(options);
  //getting value
  $('input.class-to-find').val();
  //setting value
  $('input.class-to-find').val('YYYY-MM-DD');
  //with timepicker
  $('input.class-to-find').datetimepicker()
});
Options:
{
  week_starts: [0..6] // day of week, when week starts (0 - Sunday, 6 - Saturday), default: 0
  access_past: true/false // accept or deny choose dates in past, default: true
  _default: "YYYY-MM-DD" // predefined date, default: today
}

Tags

Description:
Plugin for multiple select with autocomplete
Usage:
<input type="text" class="s-tags class-to-find" />
$(function () {
  // intitalize
  $('input.class-to-find').tags(options);
  // getting value (it returns array)
  $('input.class-to-find').val();
  // setting value
  $('input.class-to-find').val(['tags1', 'tags2', 'tags3']);
});
Options:
{
  tags: ['tag1', 'tag2'], // default selected tags default: []
  tips: ['tag1', 'tag2'], // default not selected tags default: []
  withId: true/false, // if true tags must be hash {id: string|int, name: string} default: false
  withDropdown: true/false, // if true then control will be with dropdown where user can select tag default: false
  withNew: true/flase, // if true then user can create new tag default: false
  callback: function, // callback on keyup event, it has one arg - text in control and must return array of tips default: null
  async: true/false, // if true then callback function is asynchronous and must return jQuery.Deferred object default: false
}

Mutations

Description:
Plugin for handling of DOM changes. It uses default MutationObserver or outdated events (DOMNodeInserted, DOMNodeRemoved, DOMAttrModified)
Usage:
// initialize
$('selector-of-element').mutations(options)
// event handling
$('selector-of-element').on('child_added|child_removed|attributes_changed')
Options:
{
  children: true/false, // track changes of children (add or remove) default: false
  attributes: true/false, // track changes of attributes default: false
  subtree: true/false, // if true then changes will be tracked for children and  children of children and so on default: false
  filter: [] // attributes filter default: []
}
